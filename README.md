# eda1-trabalho-pratico
CRUD de Clientes e Produtos

Projeto desenvolvido para a disciplina de Estrutura de Dados 1 na Universidade de Brasilia. O objetivo deste trabalho é desenvolver um sistema em linguagem C que utilize listas simplesmente encadeadas para armazenar e manipular dados dinamicamente, implementando operações completas de CRUD (Create, Read, Update, Delete).
